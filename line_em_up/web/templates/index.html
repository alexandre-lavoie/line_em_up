FAQ


What is Line 'Em Up?

Line 'Em Up is Tic Tac Toe reinvented. This new game adds dynamic board sizes and solve lengths, and a new "blocked" tile. This makes for a fun and unique AI challenge! The game concept was proposed by Dr. Leila Kosseim for the COMP 472 class at Concordia.


What is this Platform?

This platform is an open source web implementation of Line 'Em Up. It runs on Python/Flask and aims to be very easy to get up and running both for server and clients. This also has the benefit of being able to compete with other AIs (online or local) on the same base. This platform was made by Alexandre Lavoie for the aforementioned COMP 472 class.


How do I Play?

You can start a new game here. You can set the configurations to your liking (the default values give a base Tic Tac Toe game). You have two play options: Human or AI. For your first run, you should try playing against yourself in Human mode. You can join the game by clicking the Join button. Once you get a feel for the mechanics, you can join with an AI client.


How do I Actually Play?

You can find base Python AI here. You simply need to implement the #TODO fields and run python3 main.py client .... You can have a look at the README.md for more details about the configuration. If you want to program an AI in another language, you can have a look at the socketio calls that I made between sever and client. The API is relatively simple.


How do I Run My Own Server?

You can download the server source here. You simply need to run python3 main.py server and a server instance should start at http://localhost:5000. You can have a look at the README.md for more details about the configuration.
